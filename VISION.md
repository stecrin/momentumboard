# MomentumBoard

## Purpose

This is not a normal productivity app.

MomentumBoard is a personal recovery and execution system designed to support structured daily execution, task organisation, and decision-making during demanding periods.

The goal is not to do more.

The goal is to stabilise, simplify, and execute the few actions that actually matter.

## Current Mission

Phase 1: 30-Day Stabilisation

Main priorities:
1. Job applications
2. Rent and debt recovery
3. Selling cars/assets
4. Health and nervous system regulation
5. Emotional boundaries
6. Daily execution discipline

## System Philosophy

When life is unstable, the system must reduce complexity.

It should help me answer:

- What is the most important thing today?
- What must be ignored today?
- What is causing stress?
- What action creates money or stability?
- What boundary must I protect?
- What is the next smallest step?

## What This Is Not

This is not a fantasy productivity dashboard.

This is not for collecting endless tasks.

This is not for building a perfect life plan while avoiding real action.

This is a command system for recovery, clarity, and execution.
