# Phase 1 — Stabilisation

## Objective

Stop chaos.
Reduce overload.
Regain control.

This phase is NOT about:
- scaling
- building empires
- advanced productivity
- perfection
- complex planning

This phase IS about:
- stability
- execution
- emotional regulation
- financial survival
- clarity

---

# Core Priorities

## 1. Income Stability
- 3 job applications per day
- Focus on realistic employment
- Prioritise immediate cashflow

## 2. Financial Recovery
- Reduce debt pressure
- Resolve rent backlog
- Sell unnecessary assets
- Stop avoidance behaviour around money

## 3. Nervous System Recovery
- Sleep
- Hydration
- Gym / movement
- Reduced emotional chaos
- Reduced conflict loops

## 4. Daily Execution
Success is:
- consistency
- simplicity
- repeated action

Not intensity.

---

# Recovery Rules

## Rule 1
No fighting multiple wars at once.

## Rule 2
Reduce complexity aggressively.

## Rule 3
When overloaded:
return to:
- water
- breathing
- 3 tasks only

## Rule 4
Money problems must be faced directly.

No avoidance.

## Rule 5
Protect mental clarity from emotional destabilisation.

---

# Daily Minimum Standard

A successful day is:

- 3 job applications
- 1 financial action
- movement / gym
- one important task completed
- no emotional self-destruction
